# 🛠️ Servo - Service Marketplace Platform

Servo is a full-stack MERN application designed to connect customers with skilled service providers such as electricians, plumbers, carpenters, painters, and other professionals.The platform provides a seamless experience for both customers seeking services and providers managing service requests.

## ✨ Key Features

### Customer Features

- Secure User Registration and Login
- Browse and Search Service Providers
- View Detailed Provider Profiles
- Book Services
- Track Booking Status
- Receive Notifications
- Real-Time Provider Calling

### Provider Features

- Complete and Manage Professional Profiles
- View Incoming Service Bookings
- Accept or Reject Booking Requests
- Mark Services as Completed
- Dashboard Analytics and Insights
- Receive Real-Time Call Requests

### Real-Time Communication

- Socket.IO Integration
-  Incoming Call Popup
- Accept / Decline Call Functionality
- Ringtone Alert

## 🔐 Security & Authentication

- JWT-Based Authentication
- Protected Routes
- Role-Based Access Control
- Secure API Access



## 📊 Dashboard & Analytics

Providers can monitor:

- Total Bookings
- Pending Requests
- Accepted Services
- Completed Services
- Service Activity Analytics
